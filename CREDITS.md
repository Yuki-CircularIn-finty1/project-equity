# Credits and Asset Attribution

This file tracks all third-party assets used in Project Equity, including their sources, licenses, and attribution requirements.

## How to Update This File

When adding new assets (BGM, SE, or images), add an entry in the appropriate section below with:

- **Asset Name**: Filename in the project
- **Source**: URL or source description
- **Author/Creator**: Original creator name
- **License**: License type (e.g., CC0, CC-BY 4.0, Free for commercial use)
- **Attribution**: Any required attribution text
- **Date Added**: When added to the project
